Data Structures & Algorithms

CS503 | Semester 3 | 60 Students | Dr. Williams

Course Description

This course covers fundamental data structures and algorithms used in computer science. Topics include arrays, linked lists, stacks, queues, trees, graphs, sorting algorithms, searching algorithms, and complexity analysis. Students will implement these concepts in programming assignments and analyze their performance.

Course Modules

  • 1. Introduction to Algorithms
  • 2. Arrays and Linked Lists
  • 3. Stacks and Queues
  • 4. Trees and Binary Search Trees
  • 5. Balanced Trees (AVL, Red-Black)
  • 6. Heaps and Priority Queues
  • 7. Hashing and Hash Tables
  • 8. Graph Representations
  • 9. Graph Algorithms
  • 10. Sorting Algorithms
  • 11. Searching Algorithms
  • 12. Complexity Theory

Learning Outcomes

  • Understand fundamental data structures
  • Implement common algorithms
  • Analyze algorithm complexity
  • Solve problems using appropriate data structures
  • Compare algorithm performance
